1. 实现元素的居中的方法

1.1 水平居中

方法1：

margin: xpx auto

方法2：见1.2的方法1

1.2 垂直居中

方法1：

使用绝对定位实现：

绝对定位中left 和 right的值默认是auto，所以如果不指定left和right，则等式不满足时，会自动调整这两个值.

所以要实现元素在块内的垂直居中（包括水平），就需要先将top和bottom(left和right)设为0,再将margin设为auto即可.

.box2{
            width: 100px;
            height: 100px;
            background-color: orange;
            position: absolute;
            margin: auto;
            left: 0px;
            right: 0;
            top: 0;
            bottom: 0;
        }
